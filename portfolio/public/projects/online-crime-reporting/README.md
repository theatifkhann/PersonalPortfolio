# Online Crime Reporting System

Online Crime Reporting System is a MERN stack-based web application built to make crime complaint filing more accessible, organized, and trackable through a digital workflow.

## Overview

The project focuses on moving traditional reporting steps into a structured online interface where users can submit complaints, provide case details, and interact with a system designed for clarity and accessibility.

## Highlights

- Digital complaint filing workflow
- MERN stack architecture with React, Node.js, Express, and MongoDB
- User-focused reporting forms for structured case details
- Clean interface built for easier access to public services

## Tech Stack

| Layer | Tools |
| --- | --- |
| Frontend | React |
| Backend | Node.js, Express |
| Database | MongoDB |
| Project Type | MERN Stack Web Platform |

## Core Flow

1. User opens the reporting portal
2. Complaint details are entered into structured forms
3. The backend processes and stores the submission
4. Complaint information is managed through the application workflow
