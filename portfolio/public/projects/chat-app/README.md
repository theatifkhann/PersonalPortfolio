# Personal Mobile Chat App

A mobile-first chat application focused on realtime messaging, backend communication, and an approachable user experience.

## Overview

This app explores the core flow of modern messaging products: sending messages quickly, updating conversations in near realtime, and keeping the mobile interface clean and readable.

## Features

- Mobile-friendly chat screens
- FastAPI backend integration
- Realtime-oriented communication flow
- Structured API handling for conversation data

## Tech Stack

| Layer | Tools |
| --- | --- |
| Mobile UI | React Native |
| Backend | FastAPI |
| Communication | API Integration |
| Focus | Realtime UX |

## App Flow

```text
User sends message -> API processes request -> conversation updates in the app
```

## Learnings

- Designing around smaller mobile screens
- Organizing communication state cleanly
- Building a smoother user journey around repeated interactions
